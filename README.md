# Nifty Apes DApp

## Stacks

- React
- Chakra UI (emotion)
- Jest & React Testing Library for unit and integration test
- Storybook

## Engines

### Node

- You can install node version using nvm.
  Node = 16.11.0

### Yarn

- You can set yarn version by using this:

```shell
corepack prepare yarn@1.22.17 --activate
```

## How to install deps

```shell
yarn install
yarn run prepare
```

## How to run the app

```shell
yarn start
```
